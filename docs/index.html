02 — scrollTo()

Jump anywhere.

Click a card to smooth-scroll to that section. Supports elements, pixel values, and offset adjustments.

🚀
Hero
scrollTo(#hero)
👁
Viewport
scrollTo(#viewport)
🔒
Lock
scrollTo(#lock)
// Scroll to element with -80px offset (sticky header!)
scrollTo(document.querySelector('#section'), { offset: -80 })

// Scroll to pixel value
scrollTo(500)

scrollToTop()  // back to top
scrollToBottom() // jump to bottom
03 — isInViewport() / useInViewport()

Visibility detection.

These cards animate in as they enter the viewport. Powered by useInViewport() with IntersectionObserver.

⚡
Fast
IntersectionObserver powered
🎯
Precise
Configurable threshold
✨
Once
Trigger only on first enter
// React hook
const [ref, inView] = useInViewport({ threshold: 0.2, once: true })

return <div ref={ref} style={{ opacity: inView ? 1 : 0 }}>
  I fade in when visible!
</div>
04 — lockScroll() / unlockScroll()

Freeze the page.

Prevent body scroll when a modal or drawer is open, then restore the exact scroll position on unlock.

Status: unlocked
// When opening a modal
lockScroll()

// When closing — restores scroll position automatically
unlockScroll()
