03 — onScrollEnd()

When scrolling stops.

Fire a callback once the user has stopped scrolling for a configurable delay. Great for lazy-loading, analytics, or autosave.

⏱
Scroll, then stop.
Indicator fires after 200ms of no scrolling.
Fired 0 times this session.
// Fires once idle for `delay` ms
const stop = onScrollEnd(() => {
  console.log('User stopped scrolling!')
}, { delay: 200 })
stop()
04 — scrollIntoViewIfNeeded()

Only scroll when necessary.

Scrolls to an element only if it's outside the viewport. Click a card — if the target section is already visible, nothing happens.
